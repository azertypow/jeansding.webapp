# jensdinge

## Project Setup

### install node dependencies

```sh
npm install
```

### For production

Build and upload dist/ directory content to online server

```sh
npm run build--for-server
```


### For Development

#### Compile and Hot-Reload

```sh
npm run dev
```

#### srcript use for GitHub action for dev online production test

### Type-Check, Compile and Minify for Production on GitHub page

```sh
npm run build
```
